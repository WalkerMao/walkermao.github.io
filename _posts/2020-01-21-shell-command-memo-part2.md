### Calculation

| Syntax            | Example             |
| ----------------- | ------------------- |
| `$((expression))` | `a=$((1+1))`        |
| `` `expr expression` `` | ``a=`expr 1+1` `` |
| `$[expression]` | `a=$[1+1]` |
| `declare -i variable=expression` | `declare -i a=1+1` |
| `let variable=expression` | `let a=1+1` |
